Text Reveal & Stagger Animations

Kattints a gombokra az animációk triggereléshez. Framer Motion-ban: variants + staggerChildren

1. Karakter animáció
A talaj nem végtelen erőforrás
Framer Motion: variants={{ hidden: { y: "100%" }, visible: { y: 0 } }} + staggerChildren: 0.02
2. Szó animáció (3D rotateX)
Miért fulladnak meg a gyökerek?
Framer Motion: initial={{ y: 40, rotateX: 20 }} + staggerChildren: 0.1
3. Sor animáció (Clip Reveal)
Lazítás forgatás nélkül
Technika: overflow: hidden container + translateY(100%) → translateY(0)
4. Typewriter Effect
JS: setInterval karakterenként hozzáfűz. Framer Motion: animate={{ width: "100%" }} monospace font-tal.
5. Stagger List Items

  • Talajszerkezet megőrzése

    A forgatás nélküli lazítás nem borítja fel a természetes rétegeket.

  • Eketalp megszüntetése

    Az ásógép 45+ cm mélységig dolgozik, áttörve a tömör réteget.

  • Jobb vízgazdálkodás

    A lazított talaj képes elvezetni és tárolni a vizet.

  • Gyökérzóna bővülése

    A növények mélyebb gyökérrendszert fejleszthetnek.

Framer Motion: staggerChildren: 0.08, delayChildren: 0.2
6. Gradient Sweep Reveal
Tudományos kutatás a talajszerkezetről
CSS: background-size: 200% + background-position animáció. Gradient a szövegen áthalad.
7. Blur → Focus
Az intenzív öntözés és a nehéz gépek taposása
Framer Motion: initial={{ filter: "blur(10px)", scale: 1.1 }} → animate={{ filter: "blur(0)", scale: 1 }}
8. Scramble Text
ÁSÓGÉP TECHNOLÓGIA
JS: Random karakterek → fokozatosan cserélődnek a végleges betűkre. Ideális tech/modern feel-hez.
